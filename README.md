# Multisig-sync
Multisignature transaction sync to offchain

## Installation

```bash
$ npm install
```

## Running the app

```bash
cp .env.example .env
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
